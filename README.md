Book Tracker

**Автор:** [Альберт Шайхетдинов]

Описание
Простое приложение на Python для ведения списка прочитанных книг. 
Позволяет сохранять базу данных в JSON-файл и фильтровать список по жанрам или объему.

Основные функции
- Добавление книг (Название, Автор, Жанр, Страницы).
- Валидация ввода (запрет пустых полей, проверка числового формата страниц).
- Автоматическое сохранение данных в `books.json`.
- Фильтрация списка по минимальному количеству страниц и названию жанра.

Инструкция по запуску
1. Установите Python версии 3.x.
2. Скачайте файл `main.py`.
3. Запустите программу через терминал:
   ```bash
   python main.py
   ```

Примеры использования (Тесты)
1. **Тест добавления:** Введите "1984", "Оруэлл", "Антиутопия", "320". Нажмите "Добавить". Книга появится в списке.
2. **Тест валидации:** Оставьте поле "Страниц" пустым. Программа выдаст предупреждение.
3. **Тест фильтра:** Введите в фильтр страниц "200". В таблице останутся только книги толще 200 страниц.
